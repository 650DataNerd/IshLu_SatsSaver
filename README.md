# IshLu_SatsSaver

A Bitcoin Lightning-powered savings system designed to help users automate and manage sats accumulation with intelligence, tracking, and optional AI insights.

## Core Features
- Bitcoin Lightning integration 
- Automated savings flows
- Wallet tracking and analytics
- AI-driven savings insights (optional module)

## Stack
- FastAPI (backend)
- Lnbits(Lightning)
- Supabase (database)
- Python
- Optional AI layer for financial insights

## Goal
Make Bitcoin savings automated, trackable, and intelligent — not manual or chaotic.
